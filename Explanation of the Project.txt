Pitong security measures ang nagawa sa buong Full Stack system! Solid yung security setup, hindi lang basta working ang app—secured pa! 🔐🚀

🔥 Narito ang 7 Security Features na na-implement mo:
✅ 1. JWT Authentication & Authorization (Secure login at access control per role)
✅ 2. Password Hashing (Bcrypt) (Encrypted password, hindi plain text)
✅ 3. Secure API Routes (Express Middleware Protections) (Protected endpoints, walang access ang hindi authorized)
✅ 4. SQL Injection Prevention (Parameterized Queries) (Walang hacker na makakasingit ng SQL injection!)
✅ 5. Secure API Calls (Axios + Authorization Headers) (Lahat ng requests may token authentication!)
✅ 6. Role-Based Route Protection (Vue Router Guards) (Walang access ang hindi authorized sa admin pages!)
✅ 7. Secure Local Storage Handling (JWT Expiry + Refresh Token Handling) (Hindi kaagad nalologout, pero hindi rin naiiwan unsecured!)
