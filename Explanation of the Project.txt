🛡️ Security Features You Implemented:
✔ 1. JWT Secret Key & Refresh Token Secret Key → ✔️ Secure authentication
✔ 2. Hash Password Before Storing in DB → 🔑 Protects passwords from leaks
✔ 3. Generate Access & Refresh Tokens (JWT) → 🔄 Auto-authentication
✔ 4. Use a Database Connection Pool → 🗄️ Prevents "Client already connected" issue
✔ 5. Test Connection Before Server Starts → 🚀 Ensures DB is working
✔ 6. Prevent Unexpected DB Errors → 🔧 Stops app crashes
✔ 7. Graceful Shutdown → 🔄 Closes DB connection when server stops
✔ 8. Middleware to Verify JWT Token & Auto-Refresh → 🔄 Keeps users logged in securely
✔ 9. Secure Parameterized Queries → 🚫 Prevents SQL Injection
✔ 10. Protect Admin Routes → 🚨 Blocks unauthorized access
✔ 11. Auto-refresh Token If Missing → 🔄 Prevents forced logouts
✔ 12. Extract User Role from JWT → ⚠️ Prevents tampering
✔ 13. Role-Based Access Control (RBAC) → 🏛️ Controls user permissions
✔ 14. Attach Access Token to API Requests Automatically → 🔗 Ensures secure API calls
